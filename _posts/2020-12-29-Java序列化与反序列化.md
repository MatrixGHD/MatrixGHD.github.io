## PHP反序列化
```php
<?php
class student{
    public $name;
    public $age;
}

$a = new student();
$a->name = "tom";
$a->age = 18;
echo serialize($a);
// 序列化后：O:7:"student":2:{s:4:"name";s:3:"tom";s:3:"age";i:18;}%
```
## Java的反序列化
```java
package com.serialize;

import java.io.IOException;
import java.io.Serializable;

public class student implements Serializable { // 继承序列化类才能进行序列化
    public String name;
    public int age;

    public student(){
        System.out.println("构造方法被调用");
    }
    public student(String name, int age){ // 重载
        this.name = name;
        this.age = age;
    }


    public void sayhello(){
        System.out.printf("hello,my name is %s, and my age is %d.\n", name, age);
    }

}
```
